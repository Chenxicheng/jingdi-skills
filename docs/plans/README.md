# Plan Documents

本目录存放 Plan mode 或执行计划产生的仓库级计划文档。

## 命名规则

- 文件名格式：`YYYYMMDD-<feature-name>-v{num}.md`。
- `YYYYMMDD` 使用创建计划当天的日期。
- `<feature-name>` 使用英文 hyphen-case，表达计划对象或功能名。
- `v{num}` 从 `v1` 开始递增。

## 版本规则

- 同一 feature 的新计划禁止修改原有计划文档。
- 需要更新规划时，创建下一个版本号的新文件。
- 新版本可以引用旧版本，但必须自包含记录当前规划所需的关键上下文、决策和验证方式。
